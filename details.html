บทคัดย่อ

เป็นการพัฒนาระบบความปลอดภัยและความสะดวกสบายสำหรับผู้อยู่อาศัย โดยมีระบบการเปิดประตูที่ใช้ Keypad ในการป้อนรหัสผ่าน เมื่อป้อนรหัสผ่านถูกต้องจะสามารถเปิดประตูเข้าไปได้ เเละเมื่อผู้พักอาศัยจะออกจากห้องจะมีระบบเซนเซอร์เพื่อตรวจจับเเละเปิดประตูให้อัตโนมัติ เมื่อประตูปิดประตูทำการล็อคเเละต้องใส่รหัสผ่านใหม่อีกครั้ง

คำอธิบายโครงงาน

Intelligent Door
ระบบความปลอดภัยและความสะดวกสบายสำหรับผู้อยู่อาศัยที่ใช้ระบบเปิดประตูด้วย Keypad และ ระบบเซนเซอร์ สำหรับการตรวจจับและเปิดประตูอัตโนมัติเมื่อผู้อยู่อาศัยต้องการเข้าหรือออกจากห้อง โดยมีรายละเอียดดังนี้

ระบบ Keypad :
- ระบบนี้ใช้ในการเปิดประตูด้วยการป้อนรหัสผ่านที่ถูกต้องที่กำหนดไว้ในระบบ
- เมื่อผู้ใช้ป้อนรหัสผ่านถูกต้อง ระบบจะอนุญาตให้เปิดประตูเพื่อเข้าหรือออกจากห้อง

ระบบเซนเซอร์ :
- ระบบนี้ใช้เซนเซอร์เพื่อตรวจจับการเข้าหรือออกจากห้องอัตโนมัติ
- เมื่อผู้อยู่อาศัยเคลื่อนไหวเข้าหรือออกจากห้อง ระบบเซนเซอร์จะตรวจจับความเคลื่อนไหวและเปิดประตูให้อัตโนมัติ
- เมื่อประตูปิด ระบบจะล็อคและผู้ใช้จะต้องใส่รหัสผ่านใหม่เพื่อเปิดประตูในครั้งถัดไป

ระบบนี้ออกแบบเพื่อเพิ่มความปลอดภัยและความสะดวกสบายให้กับผู้อยู่อาศัย โดยใช้ระบบ Keypad เพื่อควบคุมการเปิดประตูด้วยรหัสผ่าน และใช้ระบบเซนเซอร์เพื่อตรวจจับการเคลื่อนไหวและเปิดประตูอัตโนมัติเมื่อมีความเคลื่อนไหว ซึ่งช่วยให้ผู้อยู่อาศัยสามารถเข้าหรือออกจากห้องได้อย่างสะดวกและปลอดภัย และระบบนี้ยังมีความสามารถในการล็อคประตูเมื่อปิดให้ความปลอดภัยเพิ่มเติมและควบคุมการเข้าถึงให้กับผู้ใช้งานอย่างเหมาะสม

ชื่อโครงงาน : Intelligent Door

ผู้รับผิดชอบโครงงาน :

2. 65070203 นางสาววรดา สาระคนธ์
3. 65070235 นางสาวสาธิตา ติวะวงศ์
4. 65070237 นางสาวสิริทัต จงศิริ

หลักการและเหตุผล : โครงงานนี้ถูกสร้างขึ้นเพื่อตอบสนองความต้องการของผู้อยู่อาศัยที่ต้องการความปลอดภัยเเละความสะดวกสบายในการเข้าออกจากสถานที่อาศัย โดยการใช้รหัสผ่านสำหรับการเปิดประตูเพิ่มระดับความปลอดภัยโดยไม่ให้คนอื่นสามารถเข้าถึงบริเวณนี้ได้โดยง่าย เเละเพิ่มความสะดวกสบายของผู้อยู่อาศัยเมื่อต้องการออกสถานที่อาศัยโดยประตูจะทำการเปิดอัตโนมัติเมื่อเดินเข้าใกล้ประตู เพื่อลดความยุ่งยากเเละเพิ่มความสะดวกในการเข้าออกสถานที่อาศัยอีกด้วย นอกจากนี้ระบบเปิด-ปิดไฟอัตโนมัติเมื่อประตูเปิดหรือปิดช่วยลดการสูญเสียค่าไฟ ซึ่งอาจเกิดขึ้นหากผู้อยู่อาศัยลืมปิดไฟเมื่อออกจากบริเวณนี้ โดยรวมโครงงานนี้เน้นการปรับปรุงความปลอดภัยเเละความสะดวกสบายของผู้อาศัยในสถานที่อาศัยด้วยการใช้เทคโนโลยีสมัยใหม่เเละเเละการสร้างสรรค์ในการเเก้ปัญหาที่เกิดขึ้นในชีวิตประจำวันของพวกเราในทางที่มีประสิทธิภาพเเละคุ้มค่า

วัตถุประสงค์ : พัฒนาระบบความปลอดภัยและความสะดวกสบายสำหรับผู้อยู่อาศัย

เป้าหมาย : ประหยัดพลังงานและเพิ่มความสะดวกสบายในการเข้าและออกจากประตู

ประโยชน์ที่คาดว่าจะได้รับ :
1. ความสะดวกสบาย: การใช้ Keypad สำหรับการใส่รหัสผ่านเข้าประตู ทำให้ไม่ต้องค้นหากุญแจหรือใช้การเปิดประตูด้วยมือ การออกจากห้องโดยไม่ต้องเปิดปิดประตูเองเมื่อเวลาเร่งรีบสามารถออกได้ทันที
2. ความปลอดภัย: การใช้ Keypad มีความปลอดภัยที่สูง โดยต้องใส่รหัสผ่าน 5 ตัวประตู ถึงจะเปิด ซึ่งทำให้ยากต่อการคัดลอกหรือปลอมแปลง และมีความทันสมัยมากขึ้นในการป้องกันการบุกรุก

ขอบเขตของโครงการ :
- ระบบการเปิดประตูด้วย Keypad ใช้รหัสผ่านในการเข้าประตู และออกโดยประตูอัตโนมัติ
- ระบบควบคุมการเปิด-ปิดไฟอัตโนมัติ

วิธีดำเนินการ :
- ใช้รหัสผ่าน 5 ตัวในการเข้าประตู และออกโดยประตูอัตโนมัติ
- การควบคุมแสงให้เปิดเมื่อกดรหัสถูกต้องและเมื่อออกจากประตูไฟจะทำการปิดให้อัตโนมัติ

ระยะเวลาในการดำเนินโครงการ :
